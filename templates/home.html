Lithum

(Web Development.)

"Lithum" gives current Columbia University students quick and easy assess to the entire list of classic readings assigned throughout the required course "Literature Humanities". Each text displayed on this site is accompanied by a simple description, as well as a link which allows users to quickly search for and purchase each title. I created this program in order to give Columbia Univeristy students an efficent way to gain relevant information about each of the 27 texts offered through "Literature Humanities", while simultaenously working to refine my frontend and backend web development skills.

Scene Maker

(Computer Graphics.)

"Scene Maker" is a simple 3D computer graphics program, which allows users to quickly generate OpenGL primatives and position them within a user-defined viewspace. This program automates the generation of 3D figures through the utilization of numerous C++ functions, which greatly reduce the amount of code needed to develop these simple 3D scenes. I created this program as a means to gain a better understanding of 3D computer graphics programs, as well as the basic foundations of modern 3D animation softwares such as Maya and Blender.

Sticky Note

(Web Development.)

"Sticky Note" is a useful study aid that allows users to create and save their own set of virtual flashcards and sticky notes. These notes can store both text and images, allowing users to save different types of information which can then be quickly accessed and altered later. Users are also given the opportunity to alter the color and size of each note they create, as a means to group information together. I developed this web application in order to give students an interactive and effective way to study online, as well as a safe and reliable way to store their course materials.

Quote Stream

(Web Development.)

"Quote Stream" is a web application that allows users to store a collection of their favorite quotes, which are then automatically iterated through and displayed to the user. Users can quickly add, update, and delete quotes from their collection, and pause the stream at any given time. Similarly, each quote is linked directly to a Google search, so users can quickly find relevant information related to their favorite quotes. I created this web application as a means to store and display my favorite literary quotes.
